# MonApp (Expo) — TP4 React Native

Fichiers créés:

- `App.js` — point d'entrée, navigation (stack + tabs), SafeAreaProvider
- `screens/HomeScreen.js` — écran d'accueil
- `screens/DetailsScreen.js` — écran de détails (reçoit param id)
- `screens/SettingsScreen.js` — écran paramètres
- `screens/AppBar.js` — barre d'application fixe
- `package.json` — manifeste minimal avec dépendances suggérées

Installation et lancement (PowerShell):

```powershell
cd c:/Users/Admin/Documents/tp
npm install
npm start
```

Ouvrir Expo Go sur ton téléphone et scanner le QR code pour lancer l'app.

Remarques et hypothèses:

- Les versions dans `package.json` sont des exemples compatibles avec Expo SDK 49. Adapte si tu utilises une autre version d'Expo.
- Si tu as déjà créé le projet avec `npx create-expo-app MonApp`, conserve cette arborescence et intègre les fichiers créés ici dans le dossier du projet.

Prochaine étape proposée:

- Vérifier l'installation des dépendances (`@react-navigation/*`, `react-native-screens`, `react-native-safe-area-context`) et tester dans Expo Go.
